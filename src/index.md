# User Guide for **BORIS App**, the **B**ehavioral **O**bservation **R**esearch **I**nteractive **S**oftware

BORIS App is a user-friendly application designed for live observations with Android.
It is free and open source, and it runs on Android.

The official BORIS website is [**https://www.boris.unito.it**](https://www.boris.unito.it).

This user guide applies to BORIS App version **{{ boris.version }}**.
